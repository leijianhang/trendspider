# A股和期货技术分析平台 - 前端

## 功能特性

- ✅ 专业K线图表 (基于TradingView Lightweight Charts)
- ✅ 股票/期货搜索和切换
- ✅ 多时间周期 (1分钟到月线)
- ✅ 复权处理 (前复权/后复权/不复权)
- ✅ 技术指标 (MA, EMA, MACD, RSI, KDJ, BOLL)
- ✅ 暗色主题UI
- ✅ 响应式布局

## 技术栈

- React 18
- Vite (构建工具)
- Ant Design (UI组件库)
- TradingView Lightweight Charts (图表库)
- Zustand (状态管理)
- Axios (HTTP客户端)

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将运行在 `http://localhost:3000`

### 构建生产版本

```bash
npm run build
```

## 主要功能

### 1. 搜索和选择品种
- 支持A股和期货切换
- 实时搜索股票/期货代码和名称
- 点击选择后自动加载K线数据

### 2. K线图表
- 专业的蜡烛图显示
- 成交量柱状图
- 支持缩放和拖动
- 十字光标跟踪

### 3. 时间周期
- 分钟级: 1分、5分、15分、30分、60分
- 日线、周线、月线

### 4. 技术指标
- **MA**: 移动平均线，支持多周期
- **EMA**: 指数移动平均线
- **MACD**: 平滑异同移动平均线
- **RSI**: 相对强弱指标
- **KDJ**: 随机指标
- **BOLL**: 布林带

### 5. 复权处理 (仅A股)
- 前复权
- 后复权
- 不复权

## 配置

### 环境变量

创建 `.env` 文件:

```
VITE_API_BASE_URL=http://localhost:3001/api
```

### 主题定制

修改 `src/styles/index.css` 和各组件的CSS文件来定制颜色和样式。

当前主题色:
- 背景色: `#0a0e27`
- 面板色: `#0f1329`
- 边框色: `#1a1e3a`
- 主色调: `#1890ff`
- 涨色: `#ef5350`
- 跌色: `#26a69a`
